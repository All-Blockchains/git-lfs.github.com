An open source Git extension for versioning large files

Git Large File Storage (LFS) is an open source Git extension for managing binary files or any file bigger than a few MB. Git LFS replaces large files such as audio samples, videos, datasets, and graphics with text pointers inside Git, while storing the file contents on a remote server like GitHub.com or GitHub Enterprise.
